Cara Top Up

  • 1. Pilih game yang ingin kamu top up.
  • 2. Masukkan User ID (dan Server ID jika diminta).
  • 3. Pilih jumlah diamond/UC/VP sesuai kebutuhan.
  • 4. Pilih metode pembayaran favoritmu.
  • 5. Klik Beli Sekarang dan lakukan pembayaran.
  • 6. Diamond akan otomatis masuk ke akunmu setelah pembayaran berhasil.
