Pilih Game

Cara Top Up

  1. Pilih game yang ingin kamu top up di halaman beranda.
  2. Masukkan User ID dan Server ID (jika diperlukan) dengan benar.
  3. Pilih nominal top up yang kamu inginkan.
  4. Pilih metode pembayaran yang tersedia.
  5. Masukkan kode voucher (jika ada) untuk mendapatkan diskon.
  6. Klik "Beli Sekarang" untuk melihat detail pesanan.
  7. Lakukan pembayaran dan kirim bukti transfer ke Admin via WhatsApp.
  8. Top up akan segera diproses oleh Admin.
